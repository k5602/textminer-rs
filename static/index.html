API Usage

You can also use the Redactr API programmatically:

Request

POST /api/redact
Content-Type: application/json

{
    "text": "Khaled Mahmoud lives in Alexandria and his email is khaled.mahmoud@example.com ."
}

Response

{
    "redacted_text": "[PERSON] lives in [LOCATION] and his email is [EMAIL].",
    "processing_time_ms": 42
}
